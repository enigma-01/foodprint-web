# Foodprint Web App

### Installation
#### 1. Clone the repo 
```
git clone https://github.com/stevennx/foodprint-web.git
```
#### 2. Install dependencies & Spin-up dev server
```
cd foodprint-web
npm i 
npm start
```
#### 3. Checkout a branch on dev!
```
git fetch
git checkout dev
git checkout -b INSERT_BRANCH_NAME
```
